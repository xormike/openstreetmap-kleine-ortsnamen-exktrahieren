# openstreetmap-kleine-ortsnamen-exktrahieren
Vollautomatische Pipeline zur Extraktion von Ortsnamen aus OpenStreetMap-Daten — komplett offline, ohne API-Schlüssel. Das Programm lädt Kartendaten herunter, schneidet einen beliebigen Ausschnitt aus, analysiert ihn und exportiert alle gefundenen Orte (Weiler, Dörfer, Stadtteile usw.) in eine übersichtliche Excel-Tabelle
